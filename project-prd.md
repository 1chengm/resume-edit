# **个人简历制作工具（MVP）核心功能说明**

## **一、项目概述**

该产品是一款在线个人简历制作与优化工具，面向求职者提供：

* 多模板简历创建与实时预览
* 简历内容优化与 AI 评分
* JD 匹配分析（岗位匹配度）
* 导出 PDF 与分享功能
* 数据统计与分析

核心原则：

1. **结构化字段脱敏**：调用 AI 时，仅发送脱敏后的 JSON，不含姓名、联系方式等敏感信息。
2. **动态系统提示词**：AI 分析和 JD 匹配的提示词存储在 YAML 配置文件，可随时更新。
3. **用户可控**：用户可循环编辑、分析并采纳建议。

---

## **二、核心模块**

### 1. **简历编辑器**

* **功能**

  * 选择多种模板（简约、创意、技术、管理）
  * 左侧表单填写结构化字段（姓名、联系方式、教育、工作经历、项目经验、技能、证书等）
  * 右侧实时预览所选模板效果
  * 富文本支持（粗体、斜体、列表、段落间距）
  * 保存至 Supabase Postgres，文件存储至 Supabase Storage

* **脱敏处理**

  * 前端/后端在调用 AI 时，删除或替换敏感字段：

    * 姓名 → 删除
    * 电话、邮箱 → 删除
    * 学校 → 替换为「某大学」
    * 公司 → 替换为「某企业」

---

### 2. **AI 简历分析**

* **功能**

  * 对脱敏后的简历进行分析，评分并给出优化建议
  * 分析维度：

    * **内容完整度**：是否包含必要模块，缺失信息
    * **结构合理性**：模块顺序、逻辑、可读性
    * **表达优化**：动词使用、量化描述、精炼语言
  * 输出示例：

    ```json
    {
      "overall_score": 0-100,
      "scores": { "content_completeness": 0-100, "structure": 0-100, "expression": 0-100 },
      "content_completeness": { "missing_sections": [...], "recommendations": [...] },
      "structure": { "recommendations": [...] },
      "expression": { "rewrite_examples": [...] }
    }
    ```

* **调用流程**

  1. 用户点击“AI 分析”
  2. 系统脱敏 JSON
  3. 后端读取 YAML 配置的 `ai_resume_analysis_prompt`
  4. 调用 AI 服务（OpenAI 或 DeepSeek）
  5. 返回 JSON → 前端展示

---

### 3. **JD 匹配分析**

* **功能**

  * 用户上传或输入 JD
  * 系统对比简历与 JD 内容，输出匹配度及差距
  * 输出示例：

    ```json
    {
      "match_score": 0-100,
      "strengths": ["技能A", "项目经验相关"],
      "gaps": ["技能B缺失", "管理经验不足"],
      "recommendations": ["补充技能B", "在项目描述突出管理经验"]
    }
    ```

* **调用流程**

  1. 用户点击“JD 匹配”
  2. 系统脱敏 JSON + JD 文本
  3. 后端读取 YAML 配置的 `ai_jd_match_prompt`
  4. 调用 AI 服务
  5. 返回 JSON → 前端展示

* **区别于 AI 简历分析**

  * **AI 分析**：关注用词、表达、结构和内容完整性
  * **JD 匹配**：关注简历内容与 JD 的匹配度，不分析用词

---

### 4. **导出与分享**

* **导出 PDF**

  * 使用 html2pdf.js 客户端渲染生成 PDF
  * 可记录导出次数与时间

* **分享链接**

  * 生成只读 UUID 链接
  * 可设置权限：公开 / 私密 / 带密码
  * 链接访问可统计浏览次数

---

### 5. **数据统计与分析**

* 简历访问次数、分享点击量
* AI 分析历史记录及评分变化
* 可用于用户优化简历

---

### 6. **安全与隐私**

* **脱敏**：AI 调用仅发送去敏后的 JSON
* **原始数据存储**：加密存储，仅在用户授权下可访问
* **删除**：用户可随时删除简历及 AI 分析记录
* **合规**：符合 GDPR 等数据保护法规

## **三、系统提示词 YAML 配置示例**

```yaml
ai_system_prompts:
  ai_resume_analysis_prompt:
    description: "简历分析，关注内容完整度、结构合理性和语言表达"
    content: |
      你是一个简历分析专家，请对输入的脱敏简历进行评分和优化建议。
      关注内容完整度、结构合理性、语言表达精炼性。
      输出严格 JSON，包含总体评分、各维度评分、内容建议、结构建议、表达优化示例。

  ai_jd_match_prompt:
    description: "JD匹配分析，关注技能和经验匹配"
    content: |
      你是一个简历与职位匹配分析专家，请对输入的脱敏简历和 JD 描述进行匹配评分。
      重点关注技能和经验是否匹配，不分析用词或表达。
      输出严格 JSON，包含匹配得分、优势、差距及优化建议。
```
